# Articles Directory

This directory is reserved for dynamically generated article pages.

## Important Notes:
- The project now uses a **dynamic article system**
- All articles are rendered by /pages/article.html?id={article_id}
- Static HTML files are no longer generated here
- All article data is stored in Supabase database
- This folder may remain empty or contain dynamically generated files at runtime

## Do not commit:
- Any article-*.html files to Git
- These files are temporary and regenerated as needed
